# App Para visualização de Dados de Matrículas da EJA

## Tecnologias:
- Python
- Plotly
- Streamlit

## Sobre a aplicação:

- Essa aplicação tem como objetivo exibir dados de matrículas da Educação de Jovens e Adultos, em uma séire histórica, a partir de 2013.
- A aplicação foi desenvolvida para fins acadêmicos e está sendo ultilizadoa em várias pesquisas em universidades da Bahia.
- A aplicação contém gráficos sobre matrículas em várias categorias: cor/raça, faixa etária e por sexo.
- Todos os dados são informações públicas retiradas do site do [Inep](https://www.gov.br/inep/pt-br/acesso-a-informacao/dados-abertos/inep-data/estatisticas-censo-escolar).
- Os dados de matrículas foram obtidos através dos censos escolares de cada ano, desde 2013.
- Outras inofrmações podem ser consultadas pelo portal de acesso à informação, pelos dados abertos, no site do [Inep](https://www.gov.br/inep/pt-br/acesso-a-informacao/dados-abertos/inep-data).


## Como acessar a aplicação:

- A aplicação pode ser acessada através do link: [app-eja-graficos](https://app-eja-graficos.streamlit.app/).
- A aplicação está publicada na plataforma Streamlit e fica em "sleep mode", é necessário ativá-la.

## Desenvolvimento:

- A aplicação foi desenvolvida em Python e algumas bibliotecas e framework: Plotly e Streamlit.
- Ainda há vários pontos de melhoria nas boas práticas e código limpo.
- A aplicação foi feita em curto prazo de tempo para suprir uma necessidade urgente: consulta acessível e mais simples aos dados de matrículas da EJA.
- O projeto foi feito no período de estágio para contribuir em várias pesquisas acadêmicas de unviversidades da Bahia, principalmente a Universidade Federal da Bahia, coordenada pela Profª. Drª. [Gilvanice Musial](https://faced.ufba.br/departamento-de-educacao-i/gilvanice-barbosa-da-silva-musial).
